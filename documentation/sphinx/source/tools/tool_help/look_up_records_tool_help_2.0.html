Summary

Provided a HUC12 layer with a selection of HUC12s on it, this tool, looks up PISCES records related to those HUC12s and returns them as a table into the ArcMap table of contents. Optional parameters can fiilter to records matching species, type, or quality criteria. This tool can be used as a way to query data about HUC12s of interest, such as sources or species presence, without generating a map, or as a tool to export human readable information from PISCES to another system.

This tool will have trouble if you select HUCs covering a large portion of the state, so limiting your selection is best.

Syntax

LookUpRecords (Zones_of_Interest, {Species_and_Groups_to_Limit_Records_to}, {Species_and_Groups_Holder}, {Limit_to_Presence_Types}, {Limit_to_Collections}, {Limit_to_Collections_Holder})

Parameter Explanation Data Type
Zones_of_Interest Dialog Reference

A HUC12 layer with an active selection on it denoting the HUC12s to be looked up, or a huc12 layer covering a small area.

There is no python reference for this parameter.

 Feature Layer
Species_and_Groups_to_Limit_Records_to (Optional) Dialog Reference

Select the species or species groups to filter the selected records. You can select overlapping sets of species and/or groups and your specified taxa will be deduplicated. Once you make a selection here, it will appear in the next box, allowing you to select more. If you select nothing, the tool will return records for any species.

This field conducts a form of autocomplete. If you type 5 or more characters into the box, it will attempt to add the species that matches to the selection. If more than one species matches, then it will warn you.

There is no python reference for this parameter.

 String
Species_and_Groups_Holder (Optional) Dialog Reference

This box allows you to uncheck species or groups you added by mistake or changed your mind about. When you uncheck the box and put the mouse focus into another area (by clicking into another box, for example), items will be removed.

There is no python reference for this parameter.

 Multiple Value
Limit_to_Presence_Types (Optional) Dialog Reference

Check the boxes of any presence types you want to find records for. If you leave this box empty, records of all presence types will be returned.

There is no python reference for this parameter.

 Multiple Value
Limit_to_Collections (Optional) Dialog Reference

Similar to the species selector, this allows you to limit the returned records to ones that have been added to various data collections. For more on what a data collection is, see the full PISCES documentation (under the heading Quality Controlled Data)

There is no python reference for this parameter.

 String
Limit_to_Collections_Holder (Optional) Dialog Reference

This box allows you to uncheck collections you added by mistake or changed your mind about. When you uncheck the box and put the mouse focus into another area (by clicking into another box, for example), items will be removed.

There is no python reference for this parameter.
